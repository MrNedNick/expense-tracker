# Expense Tracker SPA

A modern expense tracker built with Vue 3, Vuetify 3, and Pinia.

## Features

- Add and track expenses in real-time
- View a breakdown of expenses with a beautiful chart
- Fully responsive and user-friendly interface

## Technologies Used

- Vue 3
- Vuetify 3
- Pinia
- Axios
- Chart.js
